# Fragment Profile

Aplikasi Android sederhana yang mengimplementasikan **Fragment** dan **Navigation Drawer**. Proyek ini dibangun menggunakan **Java** dan menampilkan halaman profil pengguna yang mencakup informasi detail pribadi serta tautan media sosial.

## 🚀 Fitur Utama
- **Navigation Drawer**: Menu navigasi samping (hamburger menu) yang mulus untuk berpindah halaman.
- **Profile Fragment**: Halaman khusus (fragment) yang terbungkus dalam `ScrollView` agar responsif di berbagai ukuran layar. Halaman ini menampilkan:
  - Foto Profil dengan bentuk lingkaran (Circular ImageView).
  - Nama Lengkap (Raihan Adi ).
  - NIM ().
  - Tempat & Tanggal Lahir.
  - Jenis Kelamin & Alamat.
  - Informasi akun Media Sosial (Instagram, TikTok, Facebook, LinkedIn).

## 🛠️ Teknologi & Tools
- **Bahasa Pemrograman**: Java
- **Minimum SDK**: 24 (Android 7.0 Nougat)
- **Target SDK**: 36
- **Build System**: Gradle (Kotlin DSL)
- **Komponen UI/Arsitektur**: 
  - XML Layouts
  - Material Design Components (`com.google.android.material`)
  - AndroidX Navigation Component (untuk pengaturan navigasi antar fragment)
